# TC-037 — Ручной Scan for downloads без файлов показывает диалог "0 файлов"

## Предусловия
- Приложение запущено с чистыми данными (пустая Library, пустая папка загрузок).
- Открыт экран Settings.

## Сценарий (Given-When-Then)

**Given** папка загрузок пуста, в Library нет работ со скачанными файлами

**When** пользователь нажимает кнопку "Scan for downloads" в Settings

**Then** появляется диалог результата сканирования (`AlertDialog`), даже когда файлов
не найдено — сообщение указывает на 0 найденных/перелинкованных файлов
**And** Library остаётся пустой после закрытия диалога

## Проверяемые данные
| Параметр | Значение |
|---|---|
| Состояние | чистые данные, пустая папка `ao3_downloads` |

## Заметки для автоматизации
- Ключевой наблюдаемый факт риска R-05: диалог должен появиться именно при нулевом
  результате — это отличает ручной триггер от auto-триггера (silent, см. TC-038),
  который в этом случае диалог НЕ показывает.
- Не путать с P2 edge case "пустая Library" в целом (§9 P2) — здесь фокус
  исключительно на поведении диалога Scan for downloads.
