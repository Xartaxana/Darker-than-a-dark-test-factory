## 3. Таймлайн суток (эталон)

- **Вечер, T0:** гейт §1 подтверждён; Lead кладёт П10/П11/П16/П17; владелец
  кладёт борда-подкладки (П2, П5–П9, П13–П15, П18) и уходит.
- **Ночь:** heartbeat-проходы разбирают борда-события; ночная регрессия;

- **Утро, T0+12ч:** владелец делает коммиты режима A (П1, П3, П4, П12) —
  единственное «появление разработчика», имитирует рабочий пуш; НЕ трогает

- **День:** build_watch подхватывает сборки; D1/D3/D9/D11-ветки; дневной
  canary по расписанию (если включён).
- **Вечер, T0+24ч:** стоп. Разбор.

## 4. Критерии успеха (все — измеримые)

1. **Покрытие веток:** каждая строка §2 имеет свидетельство из своей правой
   колонки (артефакт/строка журнала/статус). Непройденная ветка = находка
   (по чьей стороне — фабрики или спеки — решает разбор).
2. **Permission-окна = 0** за всё окно (свидетельство оператора + пост-фактум
   `/permission-audit` окна репетиции).
3. **Эскалации точны:** всё, что ОБЯЗАНО всплыть (П2, П5-спор, П8-ответ,
   П16, П18, П19), — в `state/escalations.md`/дайджесте; ложных эскалаций
   нет (каждая строка escalations соотносится с подкладкой).
4. **Длительности воркеров:** ни одной device-сессии с серией однотипных
   Timeout-фейлов без раннего blocked (fail-fast, чек 28 OS-репо — данные
   cc_usage окна).
5. **Дайджест дня** читается человеком как полная история суток (все
   подкладки узнаваемы по дайджесту БЕЗ чтения журналов).
6. **Локи не сиротеют:** к концу окна 0 живых локов старше lock_stale.

## 5. Разбор и повторяемость

- Отчёт репетиции — `runs/REHEARSAL-<дата>.md` (свободная форма + таблица
  §2 со свидетельствами); находки — штатными артефактами (bugs/ для дефектов
  фабрики — класс test_debt/механизм — решением Lead).
- Спека повторяемая: подкладки П10/П11/П16/П17 скриптуются
  (`scripts/rehearsal_seed.py` — кандидат, НЕ обязателен для первого
  прогона: первый раз — руками Lead по таблице).
- Триггер повторного прогона: крупное изменение механизмов конвейера
  (rules.yaml/transitions/pre_steps класса «меняет маршрутизацию») либо
  слово владельца перед релизной вехой.
